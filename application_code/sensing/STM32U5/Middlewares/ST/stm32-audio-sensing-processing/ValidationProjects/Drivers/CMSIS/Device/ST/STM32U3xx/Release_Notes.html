Purpose

This driver provides the CMSIS device for the STM32U3xx product. This covers

  • STM32U356xx devices
  • STM32U366xx devices
  • STM32U375xx devices
  • STM32U385xx devices
  • STM32U3B5xx devices
  • STM32U3C5xx devices

This driver is composed of the description of the registers under “Include” directory.

Various template files are provided to easily build an application. They can be adapted to fit applications requirements.

  • Templates/system_stm32u3xx.c contains the initialization code referred as SystemInit.
  • Startup files are provided as example for EWARM©, MDK-ARM©, STM32CubeIDE©.
  • Linker files are provided as example for EWARM©, MDK-ARM©, STM32CubeIDE©.

Update history

Fifth RC Release

  • Fifth RC release of CMSIS for STM32U3xx firmware package supporting STM32U366xx/STM32U356xx/STM32U385xx/STM32U375xx/STM32U3B5xx/STM32U3C5xx devices.

Contents

  • Add/Update TSC registers for U3-2M
  • Add/Update DBGMCU registers
    • Remove DBGMCU_APB1FZR[1-2] bit fields
    • Add bits for multiple DBGMCU_APBx, DBGMCU_SR, DBGMCU_PID, DBGMCU_CID registers
  • Add SYSCFG_CFGR1_TSC_G2_IO[1,3] bit definitions in SYSCFG_CFGR1
  • Add GCC startup files for U3 2M
  • Add I2C4 instance for U3 2M
  • Add SPI4 instance for U3 2M
  • Add/Update HSP registers
  • Remove ADC DIFSEL register
  • Add missing bit fields [5-8] in flash registers
    • FLASH_SECBB1Rx, FLASH_SECBB2Rx
    • FLASH_PRIVBB1Rx, FLASH_PRIVBB2Rx
